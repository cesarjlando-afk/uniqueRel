# 📄 Relatório Mensal – Aplicativo Web

Aplicativo desenvolvido para gerar relatórios mensais de forma rápida e organizada diretamente pelo navegador.  
Ideal para registrar rondas, ocorrências, fotos e outras informações de forma prática.

---

## 🚀 Funcionalidades

- 📌 Preenchimento rápido das informações do relatório  
- 🖼️ Upload de fotos (com redimensionamento automático)  
- 📤 Exportação para PDF diretamente no navegador  
- 📱 Interface responsiva para uso em celulares  
- ⚙️ Suporte a PWA (instalação como aplicativo na tela inicial)

## 📲 Como instalar como aplicativo

1. Abra o link no navegador do seu celular  
2. Toque no menu (⋮)  
3. Selecione **“Adicionar à tela inicial”**  
4. O app será instalado como se fosse um aplicativo nativo
